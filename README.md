# HydroCrack Insight

HydroCrack Insight — прототип автоматизированной системы мониторинга и управления процессом гидрокрекинга.

## Назначение

Система предназначена для:

- мониторинга технологических параметров;
- анализа отклонений;
- аварийного реагирования;
- учета ресурсов;
- прогнозирования;
- оптимизации параметров;
- формирования отчетности.

## Стек технологий

- Python
- PyQt6
- SQLite / PostgreSQL
- SQLAlchemy
- Matplotlib
- Pandas
- ReportLab
- Pytest

python main.py
